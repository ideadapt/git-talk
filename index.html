Was heißt Versionsverwaltung?

Wie bedient man das denn?

Repository erstellen 


$ git init

Bestehende Datei mit Git überwachen 


$ git add hausarbeit.docx

Änderungen speichern und eine Notiz dazu verfassen 


$ git commit -m "erste zwei kapitel meiner hausarbeit"

Nützlich

Aktuellen Stand der Datei-Überwachung einsehen 


$ git status
